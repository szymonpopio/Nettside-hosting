Sammenligning Github og Heroku

    Pros Github

  1. Quick set up
  2. Easy deployment
  3. Very intuitive

    Pros Heroku

  1. Node.js based
  2. Quick Reload times
  3. Many add-ons

Github pages er enkelt å få satt opp og tar ikke lang tid, men nettsiden må være statisk og det kan ta litt tid før nettsiden blir oppdatert. Heroku tar lengre tid å få satt opp og kan være vanskelig å forstå, men serveren er node basert som betyr at du har mye mer frihet når du lager nettsider og det tar veldig kort tid for at nettsiden oppdateres.
