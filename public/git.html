Hvordan sette opp Github pages

For å bruke GitHub så må du først lage en bruker. Dette kan du gjøre på GitHub.com. For å gjøre det enklere kan du også laste ned GitHub Desktop på GitHub Desktop.





  1. Først når du har laget nettsiden din må du legge koden til i et repository og publishe den. Hvis du bruker github Desktop så kan du gjøre dette enkelt bare åpne GitHub Desktop og finn riktig mappe så comitter du til main nederst til venstre og så publisher du ved hjelp av knappen på midten av skjermen.






  2. Så må du finne repositoriet ditt på GitHub.com og gå inn på det og finne settings, så går du inn på "Pages" nederst til venstre.






  3. Nå som du er i innstillingene til Pages trykker du der det står "None" og velger "Main", så lagrer du det du har valgt med knappen "Save".






  4. Nå vil det komme opp en lenke som endrer farge til grønn når nettsiden er lastet opp. Ved å trykke på den kan du åpne nettsiden din eller sende URL adressen til andre.
